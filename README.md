## 功能特点
- 🌾 小麦目标检测（YOLOv8s/8n/11n/10n）
- 🍃 干旱分类识别（YOLOv8-clsyolov8n-cls/yolov8s-cls/baseline SVM/optimized SVM）
- 📊 实验数据可视化
- 🖼 图片/批量检测
- 📈 检测过程可视化
- 📝 历史记录与日志管理
- ⚡ ONNX 推理加速
